CORE DEVELOPMENT OBJECTIVES

Operating as an independent R&D group in Bryan, Texas, the center is currently executing two primary projects:

  • Basic Rocket Controller: Development of flight control logic and stabilization code for small-scale rocketry.
  • AstronomicalOS: Engineering a future-state operating system tailored for aerospace hardware systems.
